# DEVIPAY

DEVIPAY adalah aplikasi agen pulsa & PPOB berbasis Flutter.

## Fitur Utama
- Login Firebase
- Top Up Saldo
- Bonus Saldo Join WhatsApp
- Pulsa & Paket Data
- PPOB (PLN, TV, dll)

## Cara Build APK
Aplikasi dibuild otomatis menggunakan GitHub Actions.

Langkah:
1. Push kode ke branch main
2. Buka tab Actions
3. Tunggu proses build
4. Download APK
